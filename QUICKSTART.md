# 快速开始

> AI 实例如何使用本记忆系统。三步骤。

---

## 启动流程

每次新会话启动时：

```
1. 读取索引层 → 搜索与当前对话相关的节点
   search_content("关键词", "index_layer.md")

2. 定位目标节点 → 读取事实层摘要
   read_file("fact_layer.md", range: 匹配行号)

3. 返回检索结果 → 恢复上下文
   向用户返回节点摘要，附引用路径
```

## 手动检索

用户提到过去事件时，自动触发检索。检索关键词来自索引层的别名映射表。

```
用户："上次那个架构设计讨论"
  → 搜索引层 aliases："架构" → "event:三层架构确立"
  → 读事实层节点 → 返回摘要
```

## 维护

- 值得记录的事件 → 追加到 `fact_layer.md`
- 新概念 → 补全 `index_layer.md` 别名映射
- 定期提交 Git，推送到远程仓库

## 文件说明

| 文件 | 用途 |
|------|------|
| `fact_layer.md` | 只读。所有记忆的存储层 |
| `index_layer.md` | 读写。别名、时间线、权重 |
| `session_continuity.md` | 跨会话链路。新实例启动时读取 |
